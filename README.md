# Brain games

package include 5 brain-games:  
calculate, even, gcd, prime, progression.  
there are 3 rounds in each game

## Setup
```sh
npm install
```
## Start game
```sh
brain-calc
brain-even
brain-gcd
brain-prime
brain-progression
```


### game-even
